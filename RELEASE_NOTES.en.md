# Gemini OCR Capture v1.0.0 Release Notes

## Initial Release (2025-03-15)

### New Features
- High-precision OCR using Google Gemini 2.0 Flash API
- Full screen capture functionality
- Selection area capture functionality
- Multi-language support (Japanese, English, etc.)
- Simple and user-friendly interface
- Encrypted API key storage
- Error handling and logging

### Technical Details
- Built with .NET 9.0 framework
- Windows Forms UI framework
- Integration with Google Gemini 2.0 Flash API
- Settings save and load functionality
- Packaged as a single executable file

### Requirements
- Windows 10/11
- Google Cloud Platform account
- Gemini API key (requires activation and billing setup)

### Known Issues
- API requests may fail for very large images
- Recognition accuracy may decrease with certain special fonts or complex layouts

### Future Plans
- Add image preprocessing features
- Add recognition result editing functionality
- Enhance hotkey customization
- Add batch processing for multiple images

## Installation
1. Download the ZIP file
2. Extract to any location
3. Run GeminiOcrCapture.exe
